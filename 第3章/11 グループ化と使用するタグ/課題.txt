# header
ページの上部を囲む時に使われる。
ロゴ画像やページタイトル、ナビゲーションメニューが含まれる。
最初に記述するhead要素とは違う為注意。

# nav
メインのナビゲーションを囲む。
ヘッダー内に作られていることが多いのでヘッダー内に作られている場合はheaderタグで囲むことを忘れないようにする。

# article
「article」は日本語で「記事」を意味する。
その為HTMLでも記事となる部分で使われることが多く、そこだけ見ても独立して成り立つような内容にもarticleタグが使われる。
ニュースサイトやブログに使われることが多い。

# section
articleタグは独立しても意味のあるタグだが、sectionはそのひとかたまりで1つのテーマがある時に使う。
articleと少し似ている為、ニュースサイトやブログでどう使われているのかよく観察すること。

# main
Webページの中心核となる部分をmainタグで囲む。
その中には、articleタグ、sectionタグも含まれる。

# aside
本文ではない補足情報はasideタグで囲む。
メインコンテンツと関連性が低い情報を表示する時に使う。

# footer
ページの下部を囲む時に使う。
コピーライトやSNSリンクなどが含まれることが多い。

#div
意味を持たずに使われるタグ。
例えば、sectionタグで囲った後、さらにその中でグループ化させたい場合に使用するのがdivタグ。
これにより要素内でグループ化ができる。
